## 📖 Overview

**MarkPOS** is a modern Software-as-a-Service (SaaS) Point of Sale
solution designed for small and medium businesses.

### Features

-   ⚡ Fast checkout experience
-   🏢 Multi-branch management
-   👥 Role-based access control
-   📦 Inventory management
-   📊 Real-time reporting
-   ☁️ Cloud synchronization
-   📱 Fully responsive design
-   🚀 Production-ready architecture

## 🛠 Tech Stack

### Frontend

-   Next.js 15
-   React 19
-   TypeScript
-   Tailwind CSS
-   shadcn/ui

### Backend

-   Supabase
-   PostgreSQL
-   Supabase Auth
-   Row Level Security (RLS)
-   Supabase Storage

### Deployment

-   Vercel
-   Supabase Cloud

## 🚀 Getting Started

``` bash
git clone https://github.com/yourusername/markpos.git
cd markpos
npm install
```

Create `.env.local`

``` env
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
SUPABASE_SERVICE_ROLE_KEY=
```

Run the development server:

``` bash
npm run dev
```

Visit:

    http://localhost:3000
